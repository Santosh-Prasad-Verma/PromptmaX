Developer Integration

Node.js Integration Example.

Fetch your brand-aligned dynamic prompts with variables natively in Node.js using basic fetch requests.

Javascript Integration ExampleTerminal
const response = await fetch("https://promptmax-backend.onrender.com/api/v1/enhance/", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    prompt: "write customer newsletter welcome email",
    enhancement_level: "expert"
  })
});

const data = await response.json();
console.log(`Enhanced: ${data.enhanced}`);
