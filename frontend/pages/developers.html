Integration Example

Enhance prompts in 3 lines of code.

Integrate structured prompt generation into your app services using standard HTTP libraries.

Python Integration ExampleTerminal
import requests

url = "https://promptmax-backend.onrender.com/api/v1/enhance/"
headers = {
    "Content-Type": "application/json"
}
payload = {
    "prompt": "make a marketing copy assistant",
    "enhancement_level": "expert"
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()
print(f"Enhanced Prompt: {data['enhanced']}")
