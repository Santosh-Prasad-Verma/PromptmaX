Introduction

PromptX is an AI-powered prompt engineering platform that helps you enhance, analyze, and optimize your prompts for better AI outputs. Our API supports multiple AI models with automatic fallback for maximum reliability.

Quick Start

Get started with PromptX in under 5 minutes:

Step 1: Sign Up

Create a free account at promptx.dev/login. No credit card required. You'll get an API key immediately.

Step 2: Install the SDK

pip install promptx-py

Step 3: Make Your First Enhancement

from promptx import PromptX

app = PromptX(api_key="your-api-key")
result = app.enhance("Write a product description for a new AI tool")
print(result.enhanced)
print(f"Quality Score: {result.score.overall}/10")

Step 4: Explore the Dashboard

Visit the playground to enhance prompts, run A/B tests, analyze quality, and manage your history — all through the browser interface.

Using REST API

curl -X POST https://api.promptx.dev/api/enhance \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"prompt": "Write a blog post about AI"}'

Authentication

PromptX uses API keys for authentication. Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

You can also pass the API key in the request body for client-side applications.

Enhance Prompt API

POST /api/enhance

Enhance a prompt with AI-powered improvements.

Request Body

Parameter Type Description
prompt required string The prompt to enhance (max 100,000 chars)
model string AI model to use: gemini, groq

Response

{
  "success": true,
  "original": "Write a blog post about AI",
  "enhanced": "Write a comprehensive, SEO-optimized blog post about artificial intelligence...",
  "original_score": 4.5,
  "enhanced_score": 9.2,
  "improvement": 4.7,
  "model": "gemini-2.0-flash",
  "classification": {
    "category": "blog",
    "confidence": 0.85
  }
}

Analyze Quality API

POST /api/quality-heatmap

Analyze prompt quality with detailed metrics.

Response

{
  "success": true,
  "data": {
    "overall": 8.5,
    "grade": "B",
    "metrics": {
      "clarity": { "score": 9.0 },
      "specificity": { "score": 8.5 },
      "structure": { "score": 8.0 },
      "context": { "score": 8.0 },
      "constraints": { "score": 7.5 },
      "output_format": { "score": 9.0 }
    },
    "suggestions": [
      { "category": "Constraints", "issue": "Lacks specific requirements", "fix": "Add length and tone constraints" }
    ]
  }
}

Python SDK

# Usage
from promptx import PromptX

app = PromptX(api_key="your-api-key")

print(f"Quality: {result.score}/10")

JavaScript SDK

// Install
npm install promptx-js

// Usage
import { PromptX } from 'promptx-js';

const app = new PromptX({ apiKey: 'your-api-key' });
const result = await app.enhance('Write a blog post');
console.log(result.enhanced);
